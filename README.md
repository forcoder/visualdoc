# VisualDoc

🎨 可视化文档工具 - 文本转图表 + Markdown 转 PDF

## 在线访问

https://forcoder.github.io/visualdoc

## 功能

### 🎨 文本转可视化
- Markdown 文本输入
- AI 生成流程图、思维导图
- 导出 PNG / PDF

### 📄 Markdown 转 PDF
- Markdown 编辑器
- 实时预览
- 导出 PDF

## 技术栈
- HTML5 + Tailwind CSS
- Mermaid.js（图表渲染）
- marked（Markdown 渲染）
- html2canvas + jsPDF（导出）

## 本地使用

直接在浏览器中打开 `index.html` 即可使用。
